There are many words which are similar in Hindi and Telugu with same pronunciation and meaning. I am a reader of both Telugu & Hindi literature. Here are some common words in Hindi & Telugu :-

Bhojan (Hindi) : Bhojanam భోజనం (Telugu)

Tyag (Hindi) : Tyagam త్యాగం (Telugu )

Aashirvad (Hindi) : Ashirvadam ఆశీర్వాదం (Telugu)

Nirnay (Hindi) : Nirnayam నిర్ణయం (Telugu)

Samadhan (Hindi ): Samadhanam సమాధానం (Telugu)

Pravesh (Hindi) : pravesham ప్రవేశం (Telugu )

Namaskar (Hindi ) : namaskaram నమస్కారం (Telugu )

Anumati (Hindi) : Anumati అనుమతి (Telugu )

Vidyarthi (Hindi) : Vidyarthi విద్యార్ధి (Telugu)

Pustak (Hindi): pustakam పుస్తకం (Telugu )

Mahila (Hindi) : mahila మహిళా (Telugu )

Sankat (Hindi) : Sankatam సంకటం (Telugu)

Swagat (Hindi): swagatam స్వాగతం (Telugu)

Dhanyavad (Hindi) : Dhanyavdamulu ధన్యవాదములు (Telugu)

Parinaam (Hindi): parinaamam పరిణామం (Telugu)

Parichay (Hindi) : Parichayam పరిచయం (Telugu )

Aakash (Hindi): Aakasham ఆకాశం (Telugu )

Pakshi (Hindi ) : pakshi పక్షి (Telugu )

These are the few words but there are certainly many words which are commonly used in Hindi and Telugu. Because of this reason I was able to learn Hindi very quickly and being a Telugu person I was able to understand 50% Hindi very easily.
